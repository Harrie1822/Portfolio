Social Media App

This project is a feature-rich social media application with real-time chat functionality. It was built using modern technologies to ensure scalability and performance.

Technologies Used

  • React Native
  • GraphQL
  • AWS

Key Features

  • Real-time messaging
  • User authentication
  • Media sharing
  • Push notifications

Challenges

One of the biggest challenges was optimizing the app for low-latency real-time communication. We used WebSockets and GraphQL subscriptions to achieve this.
