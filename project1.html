Datorama to Power BI Migration

Migrated a large Datorama Reporting Suite to the Power BI platform, consolidating 1,100+ reports into 14 dynamic Power BI reports. Implemented Row-Level Security (RLS) and dynamic hierarchical RLS to ensure data confidentiality based on user roles.

Technologies Used

  • Power BI
  • Dataflows
  • Power Automate
  • Git Integration

Key Features

  • Consolidated 1,100+ reports into 14 dynamic reports
  • Implemented Row-Level Security (RLS) and dynamic hierarchical RLS
  • Optimized data models for faster lead times
  • Implemented incremental refresh for large semantic models

Challenges

One of the biggest challenges was managing access for 1,000+ users. We streamlined access granting and revocation using SharePoint Lists, Dataflows, and Power Automate.
